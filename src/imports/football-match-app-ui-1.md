Design a modern mobile-first web app UI for organizing casual football (soccer) matches among friends.

The app will be used mostly on phones and shared through WhatsApp links, so the interface must be extremely simple, fast, and intuitive, even for users who are not very tech-savvy.

Design style:
Clean, sporty, minimal.
Inspired by Apple Sports apps and modern mobile dashboards.

Primary colors:
Deep green #0F5132
Accent green #1DB954
Background #F7F7F7
Text dark #111111
Light gray #EAEAEA

Typography:
Inter or SF Pro
Large readable headings
Simple icons

The UI must look friendly and modern but very functional.

The design must include the following screens:

Design requirements:

• Mobile-first layout
• Touch-friendly buttons
• Minimal typing required
• Large player cards
• Status icons for payment
• Clean cards and spacing
• Works well on phone screens

The UI should look like a real production sports app, not a prototype.
